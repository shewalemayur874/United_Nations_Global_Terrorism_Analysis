# Exploratory Data Analysis of United Nation Global Terrorism

## Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on the **Global Terrorism Dataset (1970–2017)** to identify meaningful patterns, trends, and insights related to terrorist incidents worldwide.

The main objective of this project is to understand the historical trends of terrorism, identify the most affected countries and regions, analyze attack types and weapon usage, and study the human impact in terms of casualties.

This analysis helps organizations, policymakers, and security agencies make better decisions for risk assessment, resource allocation, and preventive strategies.

## Dataset Details

### Dataset Name
**Global Terrorism Dataset (GTD)**

### Time Period
**1970 to 2017**

### Dataset Source
Open-source terrorism incident dataset used for global terrorism analysis.

### Key Features

- Year of attack (`iyear`)
- Country and region
- Attack type
- Weapon type
- Number killed (`nkill`)
- Number wounded (`nwound`)
- Total casualties
- Target type
- Terrorist group name
- Success/failure of attack

### Dataset Size

A large-scale historical terrorism dataset covering worldwide terrorist incidents over 47 years.

## Approach

The project was completed using the following steps:

### 1. Data Understanding

- Loaded and explored the dataset
- Checked rows, columns, and dataset structure
- Reviewed missing values and duplicate records

### 2. Data Cleaning

- Handled missing values
- Removed duplicate records
- Converted data types where necessary
- Created total casualties feature

### 3. Exploratory Data Analysis (EDA)

Performed analysis using multiple visualizations:

- Terrorist attacks over the years (Trend Analysis)
- Top 10 most affected countries
- Attack type distribution
- Casualties analysis (Killed vs Wounded)
- Weapon type analysis
- Region-wise attack distribution
- Correlation heatmap
- Pair plot analysis

### 4. Dashboard Creation

Built a visual dashboard summarizing:

- Attack trends
- Geographic impact
- Attack methods
- Casualty analysis

## Results & Insights

### Key Findings

### 1. Terrorist Attacks Increased Over Time

There is an overall increasing trend in terrorist incidents, with certain years showing sharp spikes indicating periods of intensified terrorist activities.

### 2. High-Risk Countries Identified

A small number of countries account for a major proportion of terrorist attacks, highlighting concentrated high-risk zones.

### 3. Most Common Attack Type

**Bombing/Explosion** is the most frequent attack type, followed by armed assaults and assassinations.

### 4. Most Used Weapon Type

**Explosives/Bombs** are the most commonly used weapons in terrorist incidents.

### 5. Most Affected Region

**Middle East & North Africa** recorded the highest number of terrorist attacks.

### 6. Casualties Relationship

There is a strong positive relationship between the number of people killed and wounded during attacks.
